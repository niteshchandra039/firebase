Generate and download the I band light curves of RRab stars

This page allows you to generate the light curves of RRab stars. The light curves are generated using the ANN interpoltors that were trained in Kumar et al. (2023). The light curves are generated for the I band.



The input file should contain six columns each contains the following parameters in the given order:
Mass (M/M⊙), Luminosity (log L/L⊙), Teff (in K), hydrogen abundance ratio (X), metal abundance ratio Z, and Period (in days). A sample input file is provided here.


Accepted file formats: ".dat".
